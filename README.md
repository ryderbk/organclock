# Activity Logger (Expo)

Expo + Firebase (Auth + Realtime Database) app with:
- Bottom tabs: Home (analog clock), Log Activity, Log History, Profile
- Email/password auth
- NativeWind styling

Configuration: copy `.env.example` to `.env` and set EXPO_PUBLIC_FIREBASE_* vars.
Run with Expo Go: `pnpm install` then `pnpm start` inside `mobile-expo/`.
